# Radix Icons

This directory vendors SVG assets from `radix-ui/icons` for native primitive
controls.

- Upstream: https://github.com/radix-ui/icons
- Source archive: https://github.com/radix-ui/icons/raw/main/radix-icons.zip
- License: see `LICENSE`
- Upstream README snapshot: `README.upstream.md`

Use these SVGs for primitive affordances before drawing ad-hoc painter glyphs.
