# Primitive Anatomy

This document is the canonical part map for `finui-primitives`. The part names
follow Radix vocabulary where it fits egui, and name the egui limitation when a
web part is represented as data instead of a DOM node.

The code-side catalogue is exposed through
`primitive_part_anatomy_catalogue_output`.

Acceptance tests:

- `part_anatomy_catalogue_covers_major_radix_parts`
- `part_anatomy_catalogue_entries_have_code_symbols`

| Primitive | Public parts | Role of each part | Required tests |
| --- | --- | --- | --- |
| Dialog | Root, Trigger, Portal, Overlay, Content, Title, Description, Close | Root owns open/default/modal state. Trigger opens the layer. Portal routes content. Overlay blocks background. Content hosts title/description/action parts. Close requests dismissal. | `dialog_root_output_preserves_open_default_and_modal_contract`, `dialog_part_state_mounts_when_open_or_force_mounted` |
| AlertDialog | Root, Trigger, Portal, Overlay, Content, Title, Description, Cancel, Action | Dialog anatomy with forced modal semantics. Cancel and Action expose different focus priority and destructive semantics. | `alert_dialog_root_output_forces_modal_contract`, `alert_dialog_action_output_separates_focus_priority_and_destructive_semantics` |
| Popover | Root, Anchor, Trigger, Portal, Content, Arrow, Close | Anchor fixes geometry. Trigger opens content. Portal and Content carry side/align/collision state. Arrow tracks resolved placement. Close requests dismissal. | `popover_anchor_output_preserves_anchor_rect`, `popover_content_output_preserves_state_side_align_and_mount`, `popover_arrow_side_tracks_content_relation_to_trigger` |
| Tooltip | Provider, Root, Trigger, Portal, Content, Arrow | Provider stores delay policy. Trigger supplies accessible labeling. Content and Arrow mirror placement and force-mount state. | `tooltip_provider_output_preserves_radix_contract`, `tooltip_trigger_accessibility_output_links_open_trigger_to_content_label`, `tooltip_delay_output_respects_hoverable_content_grace_and_disable_flag` |
| HoverCard | Root, Trigger, Portal, Content, Arrow | Root owns hover/focus delay state. Trigger opens non-dismissable hover content. Content carries side/align/mount. Arrow tracks placement. | `hover_card_root_output_preserves_radix_contract`, `hover_card_content_output_preserves_state_side_align_delay_and_mount` |
| DropdownMenu | Root, Trigger, Portal, Content, Group, Item, CheckboxItem, RadioItem, Label, Separator, SubTrigger, SubContent, Arrow, Shortcut | Root owns open state. Content hosts roving focus and typeahead. Item variants expose checked/selected data. Sub parts own nested open state. | `dropdown_menu_group_arrow_and_shortcut_outputs_complete_part_contract`, `dropdown_sub_trigger_and_content_outputs_preserve_radix_submenu_parts` |
| ContextMenu | Root, Trigger, Portal, Content, Item, CheckboxItem, RadioItem, Label, Separator, SubTrigger, SubContent | ContextMenu reuses menu item anatomy but opens from pointer, touch long-press, or keyboard origin. | `context_menu_pointer_open_output_preserves_pointer_origin`, `context_menu_submenu_reuses_dropdown_submenu_contract` |
| Menubar | Root, Menu, Trigger, Portal, Content, Item, CheckboxItem, RadioItem, Label, Separator | Root owns top-level roving focus. Menu/Trigger bridge top-level navigation and opened menu content. | `menubar_state_machine_integrates_top_level_and_open_content_navigation`, `menubar_trigger_output_maps_radix_data_attributes` |
| NavigationMenu | Root, List, Item, Trigger, Content, Link, Indicator, Viewport | Root owns orientation and active item. Trigger/Content drive viewport, motion, and indicator state. Link exposes active/enabled/hovered state. | `navigation_menu_interaction_output_connects_hover_focus_open_viewport_indicator_and_motion`, `navigation_menu_viewport_output_preserves_state_orientation_and_size` |
| Select | Root, Trigger, Value, Icon, Portal, Content, Viewport, Group, Label, Item, ItemIndicator, ScrollButton, Separator | Root owns open/value state. Trigger opens content. Value/Icon mirror selection. Viewport hosts items, groups, labels, separators, indicators, and scroll buttons. | `select_part_outputs_cover_value_icon_viewport_scroll_indicator_and_group`, `select_content_output_preserves_position_side_align_and_state` |
| ScrollArea | Root, Viewport, Scrollbar, Thumb, Corner | Root defines scroll type and delay. Viewport preserves native egui scrolling. Scrollbar/Thumb expose geometry. Corner mounts when both axes are visible. | `scroll_area_native_scroll_output_preserves_egui_scroll_contract`, `scroll_area_corner_output_mounts_only_when_both_axes_are_visible` |
| Tabs | Root value, List, Trigger, Content | Header/List is rendered as trigger rects. Trigger exposes active/enabled/hovered state. Content hosts selected panel. | `tabs_keyboard_target_skips_disabled_and_respects_loop_focus`, `tabs_content_options_preserve_part_contract` |
| Accordion | Root, Item, Header, Trigger, Content | Root owns single/multiple value state. Item carries open/disabled state. Trigger toggles item. Content mounts when open. | `accordion_root_options_preserve_radix_single_and_multiple_contracts`, `accordion_keyboard_action_maps_toggle_and_header_navigation_keys` |
| Collapsible | Root, Trigger, Content | Root owns open/default/disabled state. Trigger toggles open. Content mounts when open or force-mounted. | `collapsible_root_options_preserve_open_disabled_and_theme`, `collapsible_trigger_icon_tracks_open_state` |
| Form Field | Field, Label, Control, Message | Field gives id/name/invalid context. Label associates with control. Control mirrors disabled/invalid/required. Message matches validity or server/async state. | `form_field_label_and_message_outputs_preserve_validity_contract`, `form_validation_preview_output_matches_client_server_and_async_states` |
| Checkbox | Root, Indicator | Root owns checked/default/required/disabled state. Indicator mounts from checked or force-mount state. | `checkbox_root_output_preserves_state_disabled_and_required_contract`, `checkbox_indicator_output_mounts_checked_indeterminate_or_force_mounted` |
| RadioGroup | Root, Item, Indicator | Root owns value/default/required/orientation. Item exposes checked/disabled/data state. Indicator mounts for checked item. | `radio_group_root_output_preserves_item_count_contract`, `radio_group_indicator_output_mounts_when_checked_or_force_mounted` |
| Switch | Root, Thumb | Root owns checked/default/required/form state. Thumb exposes checked geometry and data state. | `switch_root_options_preserve_form_contract`, `switch_thumb_moves_between_track_edges` |
| Slider | Root, Track, Range, Thumb | Root owns value/default/min/max/step/orientation/inversion. Track, Range, and Thumb expose geometry for renderers. | `slider_root_options_preserve_radix_contract`, `slider_part_rects_follow_value_fraction` |
| Toggle | Root | Root owns pressed/default/disabled state and maps it to `data-state`. | `toggle_root_output_preserves_radix_state_contract`, `toggle_apply_pressed_respects_disabled_and_noop_state` |
| ToggleGroup | Root, Item | Root owns single/multiple mode and selected values. Item exposes selected/disabled/orientation state. | `toggle_group_root_output_preserves_radix_group_contract`, `toggle_group_item_output_preserves_value_state_and_orientation` |
| Toolbar | Root, Button, Link, Separator, ToggleGroup, ToggleItem | Root owns orientation/loop/label. Item parts expose enabled, pressed, and orientation state. | `toolbar_root_output_preserves_radix_contract`, `toolbar_item_outputs_expose_orientation_and_disabled_contracts` |
| Toast | Provider, Viewport, Root, Title, Description, Action, Close | Provider owns duration/swipe/pause policy. Viewport owns hotkey/focus. Root exposes state/swipe/accessibility. Action and Close are focusable targets. | `toast_provider_output_exposes_radix_contract`, `toast_root_output_exposes_state_swipe_and_accessibility_contract` |
| OTP Field | Root, Input, HiddenInput | Root owns value/default/autocomplete/form state. Input exposes per-cell display. HiddenInput carries form value. | `otp_root_output_preserves_radix_contract`, `otp_hidden_input_output_carries_form_value_contract` |
| PasswordToggle | Root, Input, Button, Icon | Root owns password value and visibility. Input maps visibility to type/display. Button exposes `aria-pressed`. Icon is decorative. | `password_toggle_root_output_preserves_radix_contract`, `password_toggle_button_and_icon_outputs_track_state` |
| Avatar | Root, Image, Fallback | Root owns size. Image exposes loading status. Fallback exposes delayed fallback text. | `avatar_root_output_preserves_root_size_contract`, `avatar_fallback_output_respects_delay_and_loaded_image` |
| Progress | Root, Indicator | Root owns value/max/indeterminate state. Indicator exposes normalized value. | `progress_root_output_preserves_radix_state_contract`, `progress_indicator_output_uses_normalized_root_value` |
| Separator | Root | Root owns orientation/decorative semantics and derived size. | `separator_root_output_preserves_semantic_accessibility_contract`, `separator_line_tracks_orientation` |
| Label | Root | Root owns `for`/nested-control association, required marker, disabled/muted text. | `label_root_output_preserves_for_id_and_display_contract`, `label_root_output_supports_nested_control_association` |
| AspectRatio | Root | Root computes a bounded rect that preserves the requested ratio. | `aspect_ratio_size_fits_inside_available_bounds`, `aspect_ratio_rect_centers_root_inside_bounds` |
| VisuallyHidden | Root | Root keeps text available to accessibility while rendering outside visible bounds. | `visually_hidden_output_keeps_text_contract` |
| AccessibleIcon | Root | Root distinguishes labeled icons from decorative icons and preserves disabled state. | `accessible_icon_root_output_preserves_label_contract`, `accessible_icon_root_output_allows_decorative_icon` |
