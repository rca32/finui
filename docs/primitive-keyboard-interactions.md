# Primitive Keyboard Interactions

Keyboard behavior is tested through pure state machines today. Runtime egui input
sequence coverage is tracked separately in the gap checklist.

| Primitive | Keys | Behavior | Tests |
| --- | --- | --- | --- |
| Focus/Roving Focus | Arrow keys, Home, End, Enter, Space, Escape | Move to enabled item, wrap when configured, activate current enabled item, close on Escape | `roving_focus_output_tracks_tab_indices_and_highlight`, `horizontal_roving_focus_uses_rtl_arrow_direction`, `roving_focus_escape_requests_close_without_moving_focus` |
| Dialog/AlertDialog | Escape, Tab/Shift+Tab | Escape requests close and restore focus; modal dialog traps tab cycle | `focus_manager_targets_dialog_initial_focus_and_restores_trigger_on_escape`, `modal_inert_output_blocks_dialog_outside_input_only_when_modal` |
| DropdownMenu | ArrowUp/Down, Home, End, Enter/Space, Escape, typeahead | Move through enabled menu items, activate current item, close/return from submenus, match typed label | `dropdown_menu_roving_focus_output_skips_disabled_items_and_wraps`, `dropdown_submenu_keyboard_action_enters_and_returns_by_direction`, `menu_typeahead_match_wraps_from_current_and_skips_disabled_items` |
| ContextMenu | ContextMenu key, Shift+F10, menu keys | Keyboard origin opens at trigger center, then shared vertical menu navigation applies | `context_menu_keyboard_open_output_uses_trigger_center_for_context_key_or_shift_f10`, `context_menu_roving_focus_output_uses_shared_vertical_menu_contract` |
| Menubar | ArrowLeft/Right, ArrowUp/Down, Home, End, typeahead | Top-level horizontal navigation coordinates with opened menu content; RTL reverses horizontal movement | `menubar_state_machine_integrates_top_level_and_open_content_navigation`, `menubar_roving_focus_output_respects_rtl_arrow_direction` |
| NavigationMenu | Arrow keys, Home, End, typeahead | Move among enabled navigation items, update viewport/indicator/motion when open | `navigation_menu_next_enabled_index_uses_navigation_items`, `navigation_menu_typeahead_index_wraps_and_skips_disabled_items`, `navigation_menu_interaction_output_connects_hover_focus_open_viewport_indicator_and_motion` |
| Select | Enter/Space, Escape, Arrow keys, Home, End, typeahead | Open focuses selected item, arrows skip disabled items, Enter commits, Escape cancels, typeahead wraps | `select_keyboard_output_focuses_selected_item_when_opening`, `select_keyboard_output_commits_enter_and_cancels_escape`, `select_typeahead_match_wraps_from_current_and_skips_disabled_items` |
| Tabs | Arrow keys, Home, End, Enter/Space | Move focus by orientation, reverse horizontal movement in RTL, select automatically or manually by activation mode | `tabs_keyboard_action_respects_orientation_and_activation_keys`, `tabs_keyboard_action_reverses_horizontal_arrows_in_rtl`, `tabs_apply_keyboard_action_separates_manual_focus_from_selected_value` |
| Accordion | Enter/Space, Arrow keys, Home, End | Toggle item, move header focus, skip disabled headers, wrap when configured | `accordion_keyboard_action_maps_toggle_and_header_navigation_keys`, `accordion_keyboard_target_index_skips_disabled_and_wraps` |
| Collapsible | Enter/Space | Toggle open state from trigger | `collapsible_keyboard_action_toggles_on_enter_or_space` |
| Checkbox/Switch | Enter/Space | Toggle checked state when enabled and focused | `checkbox_apply_checked_respects_disabled_and_noop_state`, `switch_apply_checked_respects_disabled_and_noop_state` |
| RadioGroup | Arrow keys, Home, End, Enter/Space | Move to enabled item, reverse horizontal arrows in RTL, activate focused item | `radio_group_keyboard_action_tracks_orientation_and_activation_keys`, `radio_group_keyboard_action_reverses_horizontal_arrows_in_rtl`, `radio_group_keyboard_target_skips_disabled_and_respects_loop_focus` |
| Slider | Arrow keys, PageUp/PageDown, Home, End | Step, page-step, clamp, reverse for RTL/vertical/inverted geometry | `slider_keyboard_action_maps_page_home_end_and_directional_arrows`, `slider_keyboard_value_steps_pages_and_clamps` |
| Toolbar | Button activation and toggle item activation | Action helpers respect enabled state and toggle state | `toolbar_apply_action_respects_kind_enabled_and_toggle_state`, `toolbar_toggle_outputs_match_group_and_item_data_attributes` |
| Toggle/ToggleGroup | Enter/Space, item activation | Toggle pressed state; group applies single or multiple selection rules | `toggle_apply_pressed_respects_disabled_and_noop_state`, `single_toggle_group_replaces_selection`, `multiple_toggle_group_toggles_membership` |
| Toast | Viewport hotkey, Escape through close target, action activation | Hotkey focuses viewport only for matching chord; action and close are focus targets | `toast_viewport_hotkey_output_requests_focus_only_for_matching_chord`, `toast_focus_targets_output_exposes_action_and_close_focus_contract` |
| OTP Field | Text input, paste/update | Sanitizes by validation type, clamps to length, ignores disabled/read-only updates | `sanitize_otp_value_applies_validation_and_length`, `otp_apply_value_sanitizes_and_respects_disabled_readonly` |
| PasswordToggle | Button activation | Toggles visible state unless disabled/read-only | `password_toggle_apply_visible_respects_disabled_readonly_and_noop_state` |
